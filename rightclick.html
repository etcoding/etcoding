Right-click menu
Get this from GutHub

This plugin creates a right-click (context) menu on an HTML element of a web page. Small (just over 1kb compressed) and easy to use.. but see the demo.

Usage: Say you have a div where you want your context menu to appear: 
<div id="myText">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
</div>


<script type="text/javascript">
   var menu1 = [
    { name: "Make text red", action: function (element) { $(element).css("color", "red"); } },
    { name: "Make text black", action: function (element) { $(element).css("color", "black"); } },
    { name: "Make text green", action: function (element) { $(element).css("color", "green"); } },
    { name: "Reload page", action: function (element) { window.location.reload(); } },
    { name: "Open Google", action: function (element) { window.location.href = "http://www.google.com" } }
];
$("#myText").rightClickMenu(menu1);

</script>

Then add a css to style the drop-down..

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aliquam auctor sodales convallis. Duis at lobortis odio. Nulla neque ipsum, tincidunt eget fermentum vel, vestibulum sed diam. Etiam est nisl, blandit et imperdiet ullamcorper, vehicula et purus.

And done. Right-click on the text above to see it in action.
